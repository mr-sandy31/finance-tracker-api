# Finance Tracker API

## Overview
This project is a finance tracking system built using FastAPI and React.

## Features
- CRUD operations (transactions)
- Financial summary (income, expense, balance)
- REST API with FastAPI
- React frontend dashboard

## Tech Stack
- Backend: FastAPI
- Database: SQLite
- Frontend: React

## How to Run (Backend)
pip install -r requirements.txt
uvicorn main:app --reload
